# Snake Unchained 2026 — Marketing & Support Site

Public site for the [Snake Unchained 2026](https://apps.apple.com/) iOS game.

Hosted via GitHub Pages from this repo's `main` branch.

## Pages

- `index.html` — landing page
- `support.html` — support contact and FAQ (App Store "Support URL")
- `privacy.html` — privacy policy (App Store "Privacy Policy URL")

## Updating

Edit the HTML or CSS, commit, push. GitHub Pages auto-deploys within ~1 minute.

```sh
git add .
git commit -m "Update support FAQ"
git push
```

## Stack

Pure HTML + CSS. No build step, no JavaScript, no dependencies. The whole site
loads in under 30 KB total.
